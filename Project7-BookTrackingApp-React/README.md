# MyReads Project

This is a book tracking application, which gets and stores information about books selected by the user from a Udacity server.
It allows users to search books based on a limited number of keywords and select them to be stored in three different shelves of their user library: currently reading, want to read and read.

## To install and launch the project

To install and start the project, run the following command line code (with npm installed):
* npm install
* npm start
